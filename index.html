  • T: Text ("What is a text, really?")
  • E: Encoding (SGML, Unicode, XML)
  • I: Interchange, Interopability, Initiative (c.f. Bauman, Holmes)
  • C: Consortium...but also, Community, Council, Collaboration, Communication

The TEI's GitHub Repo

https://github.com/TEIC/TEI

  • 2000+ issues
  • 17,000+ comments
  • ~7 comments per issue (average)

Method

  • Use GitHub API to extract all issues and issue comments
  • Encode these in CMC
  • Merge with additional data (e.g. list of council members)
